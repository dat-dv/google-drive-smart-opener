# Google Drive Smart Opener

> A macOS application that automatically opens local Microsoft Office files in Google Docs, Sheets, and Slides via Google Drive.

## 💡 Problem

Many macOS users don't have Microsoft Office installed, making it inconvenient to open Office documents.

## ✅ Solution

Automatically imports local Office files to Google Drive, opens them directly in Google Docs/Sheets/Slides, and prevents duplicate uploads using smart file mapping.

> **Requirement:** Google Drive for Desktop must be installed and running on the local machine.
